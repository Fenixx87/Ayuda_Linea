CLASE DE ELEMENTO: 003. PLANES DE CUENTAS


Cuando se define una Cuenta de Mayor de cada Plan, se especifica opcionalmente, el nivel de Autorización que requiere  dentro del Plan para consulta y entrada de movimientos. Este nivel corresponde con las Operaciones relacionadas en los Grupos 02 y 03. Lo cual indica, que, para entrar movimientos sobre  en definición, todo Usuario interesado, debe estar autorizado sobre una operación del grupo correspondiente (03) con número mayor o igual que el nivel colocado para entrada de movimientos sobre  en cuestión.

CLASE DE ELEMENTO: 004. TIPOS DE COMPROBANTES


Para  de elemento Tipos de Comprobantes, se permite dar autorizaciones para todos los Tipos de Comprobante, cuyo primer dígito de identificación es un número específico. Para esto, se debe colocar 'x' donde x es un número. Por ejemplo, si se quiere autorizar a todos los Comprobantes que comiencen por '3' se asigna  colocando como elemento, '3x'.
