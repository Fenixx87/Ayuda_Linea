Generación Archivo XML


Esta opción permite generar el Archivo XML con el formato solicitado por el SRI, este archivo debe ser validado por el DIMM Anexo Transaccional y luego ser enviado al SRI. Este proceso se lo realiza desde la pantalla que muestra la Figura 3.49.


Figura 3.  Generación de Archivo XML.


Aquí se debe elegir la compañía o grupo del cual se va a generar la información con el año y mes. Este archivo generado se debe descargar al PC y guardarlo en un directorio específico para que luego sea validado. Ingrese la siguiente información:


Generar Archivo por:

Seleccione si desea generar el archivo de una compañía o de un grupo de compañías.


Compañía o Grupo:

Este campo varía según lo que haya escogido en el campo anterior. Seleccione la Compañía o Grupo de Compañías que desea para generar el archivo.


Año Periodo:

Digite el año y el periodo en el que desea generar el proceso. 



BOTONES

 


 



Figura 3.  Proceso de Generación de Archivo XML.


Una vez generado el archivo podrá observar la siguiente información: nombre del archivo generado, tamaño del archivo, fecha y hora en la que se generó el archivo y el botón Descargar Archivo, que permite guardar o abrir el archivo generado, mediante el mensaje que se muestra en la Figura 3.51.



Figura 3.  Generación de Archivo XML – Mensajes de Descarga de archivo.


Si selecciona la opción Abrir, el archivo se desplegará como muestra la Figura 3.51, caso contrario escoja la opción Guardar, se desplegará la ventana en la que deberá seleccionar donde va a guardar el archivo, como se muestra en la Figura 3.52.


Figura 3.  Guardar Archivo XML.


Seleccione donde se va a guardar el archivo y presione el botón Guardar, se desplegará la ventana de Descarga Completa que permitirá: abrir el archivo, abrir la carpeta donde está guardado el archivo o simplemente cerrar la ventana. El archivo que se desplegará se muestra en la Figura 3.53.



Figura 3.  Archivo XML generado.
