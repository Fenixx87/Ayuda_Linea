Condiciones de Pago


Esta opción permite definir las diferentes condiciones de pago que se han acordado con los proveedores. A cada variante se le asigna un código. Luego a cada proveedor sólo hay que indicarle el código de condición de pago con que trabaja. Aquí se indican las condiciones de descuento y de interés de mora. 


Al seleccionar esta opción se despliega en pantalla un formato como el que muestra la Figura 3.7.

Para consultar y/o actualizar una Condición de Pago ya existente, dé clic en el vínculo Select, que despliega el detalle de la Condición de Pago como muestra la Figura 3.8.

Código de Condición de Pago:

Se identifica la condición de pago con dos caracteres. Estos caracteres se asignan a los proveedores para saber cual condición de pago usan. También se puede asignar a compañías y se tomará como valor por omisión para los proveedores que no tengan asignada ninguna condición de pago.


Descripción de la Condición:

Descripción que sirve como recordatorio del uso de esta condición de pago.


Estado Registro: 

Marque el estado de la Condición de Pago: “Activo” ó “Inactivo”. Los registros inactivos no se toman en cuenta durante el proceso de pagos.



DESCUENTOS


En los datos que se ingresan para descuentos solo es posible seleccionar una entrada, con las siguientes consideraciones:


Días Validos para tomar descuento

Aquí se indica durante cuántos días a partir de la fecha del documento estará disponible el descuento por pronto pago. Este campo se debe dejar en blanco si va a utilizar un día fijo del mes para descuento.


Día fijo del mes para descuento:

Si el descuento está disponible hasta un día fijo del mes sin importar en qué día del mes se haya registrado el documento entonces puede usar este campo. Coloque aquí el día del mes hasta el cual está disponible el descuento por pronto pago. Después que el documento esté vencido ya no se tomará el descuento. Si usa la opción "Días de descuento", no puede usar "Día fijo del mes para Descuento".


Porcentaje del descuento:

Aquí se indica el porcentaje de descuento que se puede descontar del saldo de los documentos a cancelar. Este porcentaje se toma sólo si el pago se hace en los días válidos para descuento.


Importe Fijo Descuento M.L. (moneda local): 

Este campo se usa si tiene proveedores que no le ofrezcan un porcentaje de descuento por pronto pago pero sí un importe fijo de descuento en moneda local. Esta opción no es válida si se usa porcentaje de descuento y viceversa.


Importe Fijo Descuento M.E. (moneda extranjera):

Este campo se usa si tiene proveedores que no le ofrezcan un porcentaje de descuento por pronto pago pero sí un importe fijo de descuento en moneda extranjera. Esta opción no es válida si se usa porcentaje de descuento y viceversa.


Cuantos días antes de vencimiento:

Se indica cuántos días antes de que venza el descuento, se extraerá el documento para ser pagado.



INTERES DE MORA


Solo es posible seleccionar una de las siguientes opciones:


Porcentaje Anual de Interés por Mora:

Digite el porcentaje anual que se aplicará por intereses al pago en mora. 


Importe Fijo Interés M.L (moneda local):

Si el proveedor no cobra un porcentaje de interés moratorio, sino un importe fijo en moneda local sin tomar en cuenta el importe de los documentos, entonces puede usar esta opción. Esta opción no se puede usar si se usa porcentaje mensual de interés de mora.


Importe Fijo Interés M.E (moneda extranjera):

Si el proveedor no cobra un porcentaje de interés moratorio, sino un importe fijo en moneda extranjera sin tomar en cuenta el importe de los documentos, entonces se puede usar esta opción. Esta opción no se puede usar si se usa porcentaje mensual de interés de mora.


A continuación presione el botón GRABAR.
