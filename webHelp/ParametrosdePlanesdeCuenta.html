Parámetros de Planes de Cuenta


El acceso a esta opción requiere que el usuario esté autorizado a crear y/o a modificar elementos de la clase de elementos PLAN DE CUENTAS.


Esta opción permite definir los Planes de Cuentas con que trabajará el sistema. Donde puede darse de alta Planes, modificarlos, activarlos o desactivarlos. Una vez que se hayan dado de alta las Cuentas de Mayor de un determinado Plan, sólo está permitido modificar algunos de sus parámetros, ya que de no ser así se pueden crear graves inconsistencias.


Los planes de cuentas se asignan posteriormente a COMPAÑÍAS, de manera que pueden solicitarse consultas e informes consolidados de las Compañías que utilicen un mismo Plan.
