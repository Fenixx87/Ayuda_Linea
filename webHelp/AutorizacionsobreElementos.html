Autorización sobre Elementos


Permite seleccionar de la lista de clases de elementos existentes, la clase de elemento sobre la que se quiere autorizar a un usuario. Ingrese la información como se muestra en  1.9.


Figura 1.   Autorización sobre elementos


Para definir las autorizaciones sobre elementos, se requiere el ingreso de los campos que se explican a continuación.


Clase de Elementos: 

Seleccione la clase de elemento sobre la que se quiere autorizar; presionando el botón ubicado al lado derecho del campo. Para asignar autorizaciones sobre elementos servirá como apoyo el apartado de Tablas de Referencia para Autorizaciones (explicado en el manual de Convenciones). 


Elementos: 

Seleccione el elemento al que se quiere dar autorización, presionando el botón ubicado al lado derecho del campo. La lista de elementos que se despliegan, pertenece a la clase de elemento seleccionada.


Grupos Operaciones: 

Seleccione el grupo de operaciones para el que se quiere autorizar, dando clic en el botón ubicado al lado derecho del campo, que despliega la lista de grupo de operaciones relacionadas con la clase de elemento. 


Operaciones: 

Seleccione el código de operación sobre el que se quiere autorizar, dando clic en el botón ubicado al lado derecho el campo. 


Usuarios: 

Seleccione el usuario físico, al que se le asignará la autorización.


Por último presione el botón Grabar.
