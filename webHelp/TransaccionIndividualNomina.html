Transacción Individual Nómina


Se selecciona el proceso que se requiera.

Seleccionado el proceso de quincena se visualizará la siguiente información.

A continuación, se presiona el botón “Ver” en la fila del trabajador (). Se  desplegara los datos de Rol del trabajador.





Posteriormente se desplegará los Rubros (por defecto se nos muestran INGRESOS y DESCUENTOS).

Se presiona el check “Mostrar todos los Rubros”. Y consecuentemente se mostrara la información de los otros Rubros asignados al trabajador.

Se presiona Generar Rol () para poder visualizar un reporte del rol de pagos del trabajador.


A continuación, se podrá visualizar el reporte del rol de pagos.

Para ver el listado de los roles de pagos de todos o un conjunto de trabajadores se debe seleccionar los trabajadores respectivos y luego presionar el botón “Rol Detallado”, si se presiona el botón “Envío de Rol” se enviará el rol de pagos a cada trabajador seleccionado.







Se presiona el botón “Listado Resumen” para obtener un reporte resumen del proceso de nómina seleccionado.

A continuación, se podrá visualizar el reporte resumen del proceso de nómina.
