Crear Zonas


Para crear una zona siga los siguientes pasos:


  • Seleccione  de Zona, presionando el ícono (), automáticamente se despliega el estado de  de Zona.
  • En el ejemplo que muestra  1.20, se puede observar que  de Zona tiene dos niveles: CLIENTES1 y CLIENTES2, donde el código de cada nivel está dado por  de  que se definió en la opción Clases de Zona. Por ejemplo: 


En donde 1 y 5 corresponden a los dígitos que conforman el código de cada nivel.


Figura 1. Crear Zona. 


  • Digite el código de zona del primer nivel (un dígito), presione la tecla ENTER e ingrese el nombre que se asigna a la zona, a continuación presione el botón Grabar o la tecla ENTER.  


  • Para continuar con el siguiente nivel, presione el botón Cancelar para salir de esa pantalla e ingresar el código (cinco dígitos) y nombre de la zona. Se recomienda dar clic en el ícono (), que indica la flecha 1, Figura 1.20, esto permite desplegar la lista de zonas, que permitirá definir dónde se va a crear la nueva zona.


  • Una vez que se digite el código (Ejemplo: 22222), presione la tecla ENTER, para ingresar el nombre de la zona.


  • Por último, presione el botón Grabar o la tecla ENTER.


El ejemplo que se describió, se trata de zonas jerárquicas, deberá ingresar en secuencia el código y el nombre de la zona de cada nivel, hasta llegar al último nivel. Cada nivel que vaya ingresando, deberá registrar en el sistema con el botón Grabar o la tecla ENTER y continuar con el siguiente.
